## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing (catalog tiles) |
| lg | 24px | Section padding (sheet margins) |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: 
- 44px minimum touch target for interactive icons (Apple/Android accessibility standard).
- 56px bottom-sheet header height for thumb-friendly swipe.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFFFF | Background, surfaces |
| Secondary (30%) | #F4F4F5 | Cards (catalog tiles), sidebar, bottom-sheet background |
| Accent (10%) | #18181B | Primary CTAs, Cart quantity indicators, Scan button |
| Destructive | #EF4444 | Remove from cart, Clear cart actions |

Accent reserved for: 
- "Complete Sale" primary button
- "Scan Barcode" floating action or header button
- Active category tab underline
- Cart badge item count
